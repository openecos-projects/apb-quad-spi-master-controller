# Source Snapshot

This repository packages the local `ref/apb_spi_master` RTL as `ip-000021` for
catalog review.

The source baseline is the local reference directory:

```text
ref/apb_spi_master
```

Packaged RTL files:

- `apb_spi_master.v`
- `spi_master_apb_if.v`
- `spi_master_clkgen.v`
- `spi_master_controller.v`
- `spi_master_fifo.v`
- `spi_master_rx.v`
- `spi_master_tx.v`

The RTL in this repository should be treated as a local source snapshot rather
than an automatically synchronized upstream checkout.
